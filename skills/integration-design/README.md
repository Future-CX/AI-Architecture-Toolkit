# Integration Design

Create integration design documents for APIs, events, files, batches, messaging, orchestration, and system-to-system data flows.

Generated integration files belong under the private lab root in `integrations/` and use the naming format `int-0001-<data-object>-from-<source>-to-<destination>.md`.

Every create or update also maintains `integrations/_integrations-overview.md` with the name, status, description, and open-question count for each integration.
