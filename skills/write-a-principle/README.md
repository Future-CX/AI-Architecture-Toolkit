# Write A Principle

Write architecture principle documents under the consuming repository's root `principles/` folder using a preset markdown template.

When this toolkit is used as a submodule, run the helper script from the private repository root so generated files are created beside `toolkit/`, not inside it.

The helper also creates or updates `principles/_principles-list.md` as an alphabetically sorted table with `name`, `description`, and `last_updated` columns.
